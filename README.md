# Identifying Suitable PPC Methods - Supplementary Material

This repository contains the supplementary material for the publication **"Identifying Suitable PPC Methods"** (CIRP Annals 2026).

## Content

The repository provides the raw data, interview sheets, and high-resolution vector graphics of the matrices presented in the paper.

### Excel Data
*   **[Company_Interview_Sheet.xlsx](Company_Interview_Sheet.xlsx)**: The structured interview sheet used for company surveys/interviews.
*   **[General_Evaluation_of_Competence_for_Procedure.xlsx](General_Evaluation_of_Competence_for_Procedure.xlsx)**: Data related to the general evaluation of competence for the procedure.
*   **[Matrix_raw_data_step_1_and_2.xlsx](Matrix_raw_data_step_1_and_2.xlsx)**: Contains the underlying raw data used to generate the evaluation and suitability matrices for Step 1 and Step 2.

## Usage
These files are provided to support the findings of the publication and allow for reproduction or further analysis of the proposed methods.
